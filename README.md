This is a website that shows cafes and their respective characteristics. It uses a REST API developed by me at: https://github.com/Murilo-Matsuora/Curso-100DaysOfPython/blob/main/Day66/main.py.

## Features
* **Check Cafe's Features**: features such as coffee price, toilet availability, wifi availability etc.
* **Add or Delete Cafes**: this program allows the user to add a cafe or delete existing ones.
* **Search**: you can also search a cafe by name.

## Project Structure
* **Backend**: Python, Flask
* **Frontend**: Jinja, Bootstrap 5, CSS
* **Data Storage**: The API database uses SQLAlchemy
